<h2>Features</h2>

<CardGrid>
  <Card title="Simple" icon="rocket">
    Publish a data package extension in minutes, without the need for coding. Generate Python and TypeScript SDKs automatically.
  </Card>
  <Card title="Modern" icon="document">
    The extension template use the latest version of the Data Package Standard (v2), allowing to implement basically any business domain logic.
  </Card>
  <Card title="Standard" icon="open-book">
    The extension template is based on the widely-used Data Package standard, making it compatible with its software ecosystem.
  </Card>
  <Card title="Automatic" icon="laptop">
    The extension template generates Python and TypeScript SDKs, simplifying the end-user experience of publishing and consuming data packages.
  </Card>
</CardGrid>
